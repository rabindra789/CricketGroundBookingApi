# Cricket Ground Booking API

Backend API for a cricket ground booking platform built with ASP.NET Core Web API and SQL Server.

## Tech Stack

- ASP.NET Core Web API
- Entity Framework Core
- Ms SQL Server
- JWT Authentication
- Swagger
